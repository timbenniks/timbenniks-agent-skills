# Tim Benniks Agent Skills

This folder contains the Tim Benniks Agent Skills repository. Each subfolder is a skill package meant to be lifted into its own distribution when ready.

## Current skills

### openapi-typescript-sdk-generator

Generates a production TypeScript SDK from an OpenAPI 3.0/3.1 spec (URL or local file). It uses human-friendly method names, requires explicit confirmation of defaults, and produces deterministic output for either a drop-in folder or a standalone package.
